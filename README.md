Wrapper for Gurobi
==================

<http://www.gurobi.com>

macOS / OS X

```
$ brew install cmake perl
$ cpan -T HTML::Form LWP LWP::Protocol::https Tk
```

Debian / Ubuntu

```
$ sudo apt-get install build-essential cmake libhtml-form-perl liblwp-protocol-https-perl libwww-perl perl perl-tk
```

All Platforms

```
$ mkdir build && cd build
$ cmake ..
$ make install
```
